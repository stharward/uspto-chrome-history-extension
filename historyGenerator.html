Select History Entries to be Printed

Instructions

This extension allows you to create a “printable” web search history that can be converted to PDF and placed into e-Red Folder so that it gets added to IFW with an Office action.

Creating the “Printable” History

  1. In the list above, uncheck the searches that you do not want to be shown on the search history listing that goes into IFW (e.g. mistakes, or searches unrelated to the application).
  2. Click the “Generate Printable Search History” button.
  3. If you need to generate multiple separate search history listings (e.g. you have searches for two applications mixed together), generate one, then reload this page, and generate the second one.

Putting the Search History in e-Red Folder and IFW

  1. When the printable search history is generated and the “Print” dialog comes up, select “Save as PDF” as the desination.
  2. Save the PDF into the e-Red Folder for the application.
  3. In OACS, import the PDF from e-Red Folder using the SRNT document type.
