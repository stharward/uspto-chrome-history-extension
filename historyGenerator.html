Instructions

This extension allows you to create a “printable” web search history that can be converted to PDF and placed into e-Red Folder. Once the history is in eRF, you can import it into an Office action with OACS so that it gets added to IFW.

Creating the “Printable” Search History

  1. In the list above, check the searches that you want to be shown on the search history listing that goes into IFW.
  2. Click the “Generate Printable Search History” button. The checked entries will appear on the printable page.
  3. If you need to generate multiple separate search history listings (e.g. you have searches for two applications mixed together), generate one, then reload this page, and generate the second one.

Putting the Search History in e-Red Folder and IFW

  1. When the printable search history is generated and the “Print” dialog comes up, select “Save as PDF” as the desination.
  2. Save the PDF into the e-Red Folder for the application.
  3. In OACS, import the PDF from e-Red Folder using the SRNT document type.

Version History and ChangeLog

Version 3 (28 Feb 2017)

  • Added a function to filter out queries that were executed, but weren’t displayed long enough to be reviewed. For example, if you realized that you misspelled a search term, and immediately ran a new query with the new term, you can filter out the old query. This is particularly useful when building a complicated query in Google Patents, because Chrome runs a new query each time you enter a new search term, and you probably don’t care about keeping all the intermediate queries that were run while you were building up the whole query.

Version 2 (1 Oct 2015)

  • Updated to handle the July 2015 overhaul of the Google Patents interface. The old interface is still around, so this extension still handles it for now.

Version 1 (29 May 2015)

  • Initial release. Supports Google Books, Images, Patents and Scholar.
